# `prophet-cli`

```
Usage: prophet-cli.py [OPTIONS]

  Command line interface for Facebook Prophet forecasting procedure.

Options:
  --infile TEXT      File to read input from. Stdin if not specified.
  --outfile TEXT     Save forecast figure to this file.
  --periods INTEGER  Number of periods (days) ahead to forecast.
  --demo             Run the Peyton Manning Wikipedia demo forecast.
  --help             Show this message and exit.
```
